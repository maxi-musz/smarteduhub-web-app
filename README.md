# SmartEduHub

SmartEduHub is a modern educational platform designed to streamline learning, collaboration, and resource management for students and educators.

## Features

- User authentication and role management
- Course creation and enrollment
- Assignment submission and grading
- Real-time notifications
- Discussion forums and messaging
- Resource sharing and document management

## Technologies Used

- Frontend: Next.js
- Backend: Node.js, Express.js/Nest.js
- Database: MongoDB
- Authentication: JWT
- Styling: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v14+)
- pnpm or yarn
- MongoDB

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/smarteduhub.git
   cd smarteduhub
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Set up environment variables in a `.env` file.

4. Start the development server:
   ```bash
   pnpm run dev
   ```
